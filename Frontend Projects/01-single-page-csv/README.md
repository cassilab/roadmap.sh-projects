# Single Page CV

A lightweight, single-page CV built entirely with HTML. This project uses semantic HTML to create a clean, accessible résumé without relying on CSS frameworks, JavaScript, or build tools.

## Project URL

This project was completed as part of the roadmap.sh Frontend Projects.

- **Project:** https://roadmap.sh/projects/single-page-cv

## Project Structure

```text
/
├── index.html
└── assets/
    └── favicon.ico
```

> **Note:** The `assets` folder only contains the website favicon. The application itself is completely self-contained within `index.html`.

## Getting Started

1. Clone or download this repository.
2. Ensure the `assets` folder remains in the same directory as `index.html`.
3. Open `index.html` in any modern web browser (Chrome, Edge, Firefox, or Safari).
4. That's it—no installation, dependencies, or build process required.

## Requirements

* A modern web browser.
* No additional software or internet connection required.

## Features

* Semantic HTML structure
* Single-page layout
* Lightweight and portable
* No CSS frameworks or JavaScript
* No build tools or dependencies
